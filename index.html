How the System Thinks

1. Observe

Capture basic usage: workbook uploads, generated outputs, citations opened, exports, corrections, reviewers.

2. Infer

Notice when similar work repeats: earnings updates, diligence, reporting, board prep.

3. Score

Grade adoption, trust, saved-preference quality, repeat use, and stakeholder spread.

4. Intervene

Route stalled accounts to concrete CS moves instead of generic check-ins.

5. Expand

Use repeated workflow proof to move into adjacent teams, reviewers, or workflows.

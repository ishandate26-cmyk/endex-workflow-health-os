Why the two views are different

Endex internal view

CS sees rule failures, account actions, and whether the second cycle proved the workflow is sticky.

Customer Excel view

The user only sees the immediate decision: apply the saved setup, review exceptions, or start fresh.

What gets recorded

If the user corrects the same item again, CS gets a precise follow-up instead of a vague health score.

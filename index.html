From Excel Nudge to CS Follow-up

Read each card left to right. The point is not to track vague engagement; it is to see whether Endex made the next real Excel run easier.

1. User starts

A real workbook or source folder opens.

2. Endex offers

Reusable setup is separated from judgment calls.

3. User chooses

The analyst accepts, edits, rejects, or starts blank.

4. CS follows up

The response turns into a specific account action.
